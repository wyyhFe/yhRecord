## 2026-06-15 · 打卡模块 V2 全量升级

### P0：热力图日历 + 心情标签

**数据库**
- 新增 `biz_checkin_tag` 标签定义表 + 20 个系统预设标签
- 新增 `biz_checkin_record_tag` 记录-标签关联表
- `biz_checkin_record` 新增 `mood` 字段

**后端**
- 新增 `GET /checkin/heatmap` 热力图接口（按月统计 + 连续天数）
- 新增 `GET /checkin/tags`、`POST /checkin/tags` 标签 CRUD
- `CheckinRequest` 扩展 `mood` + `tagIds`
- `CheckinTaskVO` 扩展 `mood` + `tagNames` + `remark` + `mediaPaths`

**前端**
- 新增 `heatmap-calendar` 热力图组件（色阶展示 + 月份切换 + 点击跳转）
- 新增 `mood-picker` 心情选择器（10 个 Emoji）
- 新增 `tag-picker` 标签多选器（预设 + 自定义）
- 打卡弹窗集成心情/标签/备注/图片

### P1：勋章系统 + 微记录 + 补卡机制

**数据库**
- 新增 `biz_achievement_medal` 勋章定义表 + 12 个预设勋章
- 新增 `biz_user_medal` 用户勋章解锁记录表
- `biz_checkin_record` 新增 `is_mend` 补卡标记字段

**后端**
- 新增 `MedalService` 勋章服务（列表 + 打卡后自动判定解锁）
- 新增 `GET /checkin/medals` 勋章列表接口
- 新增 `POST /checkin/records/mend` 补卡接口
- 新增 `GET /checkin/mend-remaining` 补卡次数接口
- `CheckinServiceImpl.checkin()` 打卡成功后触发勋章检查

**前端**
- 新增 `medal-card` 勋章卡片组件
- 新增 `medal-unlock-popup` 解锁动效弹窗
- 新增 `pages/checkin/medals.vue` 勋章陈列墙页面
- 打卡弹窗 textarea 改为 50 字限制（微记录）
- 热力图下方增加勋章入口 + 补卡次数显示
